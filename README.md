# Retail Business Performance & Profitability Analysis

## Project Overview
This project analyzes retail transactional data to identify profit-draining categories, regional
sales trends, and seasonal product behavior. The project combines SQL, Python, and Tableau
to perform data cleaning, exploratory data analysis (EDA), and interactive dashboard
visualization.

## Objectives- Analyze retail sales and profitability- Identify loss-making products and categories- Understand monthly sales trends- Compare regional sales performance- Build an interactive Tableau dashboard- Generate business insights for better decision-making.

## Tools & Technologies: SQLite- Python- Pandas- Matplotlib- Seaborn- Tableau Public- Jupyter Notebook
--
## Project Workflow
1. Data Import & Cleaning using SQLite
2. SQL Query Analysis
3. Exploratory Data Analysis (EDA) using Python
4. Correlation Analysis & Heatmaps
5. Tableau Dashboard Creation
6. Business Insights & Recommendations
